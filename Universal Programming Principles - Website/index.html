Basic Coding Concepts





What is Computer Programming?

A computer is a device that processes and stores information. Computer programming is writing a sequence of
instructions that has been translated into commands a computer can understand. These instructions are called code.


Why is coding useful?

Coding is an important skill to learn because it will be used to run and program many machines in the future. It can
also help you develop problem solving skills, and help you improve in math, writing, reading, creativity, and art. Coding
can be used to create games, animations, program robots and machines to perform different tasks, and more!
