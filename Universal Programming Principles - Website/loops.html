Loops


Loop statements allow you to easily repeat a portion of code many times. Loops save
programmers time, and also make programs shorter.


Different Types of Loops:

- For Loops: A loop that repeats itself a specific number of times. For loops are used
when you know the number of times you want something to repeat.




- While Loops: A loop that will run forever until a certain condition is met. While loops
are used when you don't know the exact number of times you want something to repeat, and
they depend on another factor.




- Nested Loops:When one loop is put inside of another loop. Nested loops help create more
complex repeating code. The inner loop has to run through all its sequences before the outer
loop code is run again.
