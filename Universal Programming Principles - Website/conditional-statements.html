Conditional Statements



What are Conditional Statements?

Conditional statements are commands for handling decisions. They only run when a certain condition
is met, and can change depending on different decisions.

Conditional statements are written in the "if... then" format, (If a specific thing happens, then this
will happen).

Examples:
  • If you touch the lava, then your character dies.
  • If you catch a coin, then your score increases by 1 point.
  • If you answer correctly, then ask the next question

If your condition is false, you can have a different action happening by using "else", and describing the
different action.

For Example:
  • If you touch the lava, then your character dies, else your character goes to the next level.
  • If you catch a coin, then your score increases by 1 point, else you loose 1 point.
  • If you answer correctly, then ask the next question else game over.

You can also combine multiple conditional statements to create more than one actions, by using "else if" in an
if statement.
