Events


What are Events?

Events are actions that cause something to happen in a program. For Example:

  • Clicking a sprite
  • Pressing a key
  • Tapping a touch screen
  • Clicking a mouse


Events can be external actions from the user, like a mouse click. They can also be
internal actions that are triggered when something happens in a program.

For Example:

  • When a character dies, the program brodcasts the message "Game Over"
  • When the music stops, stop all the scripts
  • When the player recieves ice cream, end the game
